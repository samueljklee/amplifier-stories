The Origin

What is Superpowers?

An agentic skills framework and software development methodology created by Jesse Vincent (obra). A complete workflow built on composable "skills" that enforce TDD, design-first thinking, and subagent-driven development.

25k+ Stars
One of the most popular AI development methodologies on GitHub.
Composable Skills
Markdown-based process documentation that can be mixed and matched.
Battle-Tested
Used across Claude Code, Codex, OpenCode, and now Amplifier.
The Integration

Import, Don't Duplicate

This bundle wraps superpowers skills as Amplifier agents.
Zero duplication. Automatic updates. Single source of truth.

~500 lines of wrapper code vs 20,000+ if we duplicated everything
Architecture

Git Submodule + Thin Wrappers

# Superpowers is included as a git submodule amplifier-bundle-superpowers/ ├── superpowers/ # Git submodule (obra/superpowers) │ └── skills/ # Original superpowers skills ├── agents/ # Thin wrappers (~30 lines each) │ ├── design-refiner/ │ │ └── AGENT.md # References @superpowers/skills/... │ └── ... └── bundle-dev.md # Main bundle composition

Wrappers reference skills via @mention, adapt for Amplifier tools, and stay synchronized.

Available Agents

Superpowers Skills as Amplifier Agents

Agent Purpose
superpowers:design-refiner Interactive design refinement using Socratic method
superpowers:plan-writer Create detailed implementation plans with bite-sized tasks
superpowers:tdd-enforcer Enforce RED-GREEN-REFACTOR cycle
superpowers:subagent-orchestrator Execute plans with fresh subagent per task
superpowers:code-reviewer Systematic code review focusing on correctness
superpowers:branch-finalizer Verify tests, present merge/PR options

Getting Started

Two commands to start

# Option 1: Run directly from GitHub amplifier run --bundle git+https://github.com/ramparte/amplifier-bundle-superpowers@main:bundle-dev.md "Build a feature" # Option 2: Clone for local use git clone --recursive https://github.com/ramparte/amplifier-bundle-superpowers amplifier run --bundle ./amplifier-bundle-superpowers/bundle-dev.md "Build a feature"
Pro tip: Use --recursive when cloning to get the superpowers submodule
