Chapter 6

What I Learned

Technical Insights
  • Auth is complex ‚Äî Device Code vs Client Creds vs ROPC all have trade-offs
  • Permissions are iterative ‚Äî expect multiple grant cycles
  • Licensing is hidden ‚Äî E5 required for Teams features
  • MFA adds friction ‚Äî automation accounts need exclusion
  • Token caching matters ‚Äî asyncio.Lock prevents race conditions
Development Patterns
  • Vision first ‚Äî describe what you want, not how to build it
  • Let AI architect ‚Äî zen-architect designs better than I would
  • Test in isolation ‚Äî shadow environments save sessions
  • Iterate on permissions ‚Äî expect the cycle
  • Trust the delegation ‚Äî agents know their domains
