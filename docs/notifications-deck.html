Notification Methods

Desktop, mobile, or both

Desktop Notifications
Native OS notifications via terminal escape sequences. Zero dependencies. Works over SSH.
amplifier notify desktop --enable
Push Notifications (ntfy.sh)
Push to your phone via ntfy.sh. Perfect for stepping away from your desk.
amplifier notify ntfy --enable

Enable both for maximum coverage. One command each.

Security

Privacy-first design
