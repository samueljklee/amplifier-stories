Before You Start

What you'll need

Python 3.11+
Modern Python required. Check with python --version
API Key
Anthropic recommended (Claude). OpenAI, Azure, or Ollama also work.
5 Minutes
That's all it takes. Seriously.
Step 1: Install

Two simple commands

# Install UV (Python package manager) curl -LsSf https://astral.sh/uv/install.sh | sh # Install Amplifier uv tool install git+https://github.com/microsoft/amplifier
That's it. Amplifier is installed globally and ready to use.
Step 2: Configure

Set up your AI provider

# Interactive setup wizard amplifier init # Follow prompts to: # - Choose provider (Anthropic, OpenAI, Azure, Ollama) # - Enter API key # - Select default model
Your settings are saved to ~/.amplifier/settings.yaml
Step 3: Start Chatting

Launch your first session

# Start interactive session amplifier # You'll see the prompt: > # Try your first command: > List the files in the current directory # Amplifier uses tools to interact with your system
Congratulations! You're now running an AI agent framework.
Essential Commands

What you need to know

  • / /help — Show available commands
  • / /tools — List available tools
  • / /agents — Show available agents
  • / /status — Display session status
  • / exit or Ctrl+C — End session gracefully
Type any of these at the prompt to use them

FAQ

Common first questions

How do I switch providers?
Edit ~/.amplifier/settings.yaml or run amplifier init again
Where are sessions saved?
~/.amplifier/projects/<project-slug>/sessions/ — Full conversation history in JSONL format
How do I add tools?
Use bundles! amplifier bundle add git+https://github.com/org/bundle-name
Can I use it offline?
Yes! Configure Ollama as your provider for local models
What about cost?
Depends on your provider. Track with --verbose to see token usage
Is my data safe?
Everything stays local except API calls to your chosen provider
