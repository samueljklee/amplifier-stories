Features

Smart filtering, your rules

VIP Senders
Messages from your VIP list always get through. Boss, spouse, on-call team - you decide.
Priority Keywords
"urgent", "outage", "ASAP", your project names - anything that signals importance.
Noise Detection
Automatically filters reactions, "liked your message", read receipts, and other low-signal noise.
Periodic Digests
Everything filtered goes into hourly or daily summaries. Nothing is lost, just deferred.
App Muting
Completely silence specific apps during focus time. One command to toggle.
Agent Toasts
When something surfaces, the toast explains why: "VIP sender" or "Contains 'outage'".
Usage

Simple CLI, conversational config

# Start the daemon attention-firewall run # Add a VIP (their messages always get through) attention-firewall add-vip "Alice Chen" # Add priority keywords attention-firewall add-keyword "outage" "ASAP" "prod-down" # View your digest attention-firewall summary --hours 24 # Check system readiness attention-firewall check

Or just talk to it in an Amplifier session: "Add my manager to VIPs"
