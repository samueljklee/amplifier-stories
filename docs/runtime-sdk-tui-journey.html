The Standard

Agent Client Protocol (ACP)

What is ACP?
  • Standardized protocol for IDE ‚Üî AI agents
  • Created by Anthropic, adopted by Zed, JetBrains
  • JSON-RPC 2.0 based messaging
  • Bidirectional capability negotiation
  • Streaming updates via notifications
Protocol v2025-01-07
Key Methods
initialize Handshake & capabilities
session/new Create agent session
session/prompt Send user message
session/update Streaming responses
session/cancel Stop execution
Official SDK: agent-client-protocol>=0.7.1 ¬∑ Spec: agentclientprotocol.com

Chapter 3: The Solution

TUI Tester Bundle

AI-assisted visual testing for terminal interfaces.

Capabilities
  • ‚úì Spawn - Launch TUI in headless pseudo-terminal
  • ‚úì Drive - Send keystrokes (arrows, enter, special keys)
  • ‚úì Capture - Get text, ANSI, and PNG screenshots
  • ‚úì Analyze - AI vision for visual issues
  • ‚úì Headless - No X11/display required (CI/CD)
# Spawn a TUI application result = tui_terminal( operation="spawn", command="amplifier-tui run" ) session_id = result["session_id"] # Send keystrokes tui_terminal( operation="send_keys", session_id=session_id, keys="/help{ENTER}" ) # Capture screenshot for analysis capture = tui_terminal( operation="capture", session_id=session_id ) # ‚Üí {text, ansi, image_path}
